Machine Learning project that predicts renewable wind energy outcomes using structured data and multiple ML models.

---

## Project Overview

This project builds an end-to-end machine learning pipeline for predicting wind energy patterns using historical dataset features.

The workflow includes:

- Data loading
- Exploratory data analysis
- Feature engineering
- Model training
- Model evaluation
